  1. Typography

    2. Primary Font

    • Font Family: Dosis (Sans-Serif)

    • Use Case: headings

    Secondary Font

    • Font Family: Bitter (Serif)

    • Use Case: paragraph text

    • Font Sizes

      • h1: 50px

      • h2: 48px

      • Body Text: 18px for paragraphs; 22px for schedule text

  3. Components

    Header Image

    • Full-width to cover the entire viewport width.

    • Usage: Banner images at the top of main sections.

    Header Text

    • Centered and padded for emphasis.

    • h1 at 50px and p text at 28px for subheadings.

    • Max-width: 600px to maintain readability.

    Navigation Bar

    • Style: Horizontal navigation with centered, spaced items

    • Background Color: Primary Burgundy (#703640)

    • Link Colors: White, with hover states in Dark Red (#562931)

    • Responsive Behavior: Switches to vertical stacking on screens under 800px.

    • Hover: Darkens to signal active state

    5. Flexbox

    6. Tablet and Mobile
